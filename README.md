# Astro with React
Astro + Tailwind CSS + React の静的サイト

## はじめる
1. 依存関係のインストール
```
npm i
```

2. 開発サーバーを起動
```
npm run dev
```

## デプロイ
AWS S3 + CloudFrontの構成でデプロイする。
